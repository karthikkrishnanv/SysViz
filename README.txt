SysViz: Collect and Vizualize system level data of a cluster
Requirements:
  Vanilla tools (systat,sar,iostat,awk,sed etc) on all the nodes of cluster
  "R" needs to be installed on the vizualizer client (http://www.r-project.org/)
  "R" packages ggplot2,plyr,plotrix also needs to be installed on client
  "R": You may have to build from sources
